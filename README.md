# 本格二人麻雀 (1v1 Riichi Mahjong)

Pygameを使用した、コンピュータと1対1で対戦できる本格的な二人麻雀ゲームです。リアルな牌画像、正確な和了判定、詳細な役名表示を備えています。

## 特徴

- **リアルなグラフィック**: 高品質な麻雀牌画像を自動ダウンロードして使用。
- **フルルール対応**: リーチ、ポン、カン（暗槓・明槓）、ドラ、裏ドラを完全実装。
- **正確な判定**: 七対子や三暗刻などの複雑な役も正確に判定。
- **日本語表示**: 上がった際の役名を正式な日本語でリスト表示。
- **賢いAI**: シャンテン数（上がりまでの最短距離）を計算して打牌する思考ルーチン。

## 二人麻雀ルール

このゲームでは、テンポの良い対局のために以下のルールを採用しています。

1. **使用牌 (計108枚)**:
   - **萬子**: 1・9のみ（2〜8は使いません）
   - **筒子**: 1〜9
   - **索子**: 1〜9
   - **字牌**: 東・南・西・北・白・發・中
2. **和了条件**:
   - 4面子1雀頭、または**七対子**（同じ牌2枚 × 7組）を揃える。
   - 役が1つ以上あること（リーチやツモも役に含まれます）。
3. **鳴き**:
   - **ポン・カン**が可能（チーはありません）。
4. **リーチ**:
   - テンパイ時に1000点を支払って宣言可能。以降は自動ツモ切りとなります。
5. **ドラ**:
   - ドラ表示牌の次がドラとなります。リーチして上がった場合は**裏ドラ**も計算されます。
   - カンをすると新ドラが追加されます。

## 実行方法

Python環境と `uv` がインストールされている場合、以下のコマンドで起動できます。初回起動時に画像アセットが自動的にダウンロードされます。

```bash
uv run main.py
```

## 操作方法

- **打牌**: 自分の手番（牌がハイライトされている時）に、捨てたい牌をマウスでクリックします。
- **アクション**: ロン(RON)、ツモ(TSUMO)、リーチ(RIICHI)、ポン(PON)、カン(KAN)ができる場合は、画面右側にボタンが出現します。
- **効果音**: スコア欄の SOUND ボタン、または **Mキー** でオン/オフを切り替えられます。
- **リスタート**: 対局終了後、**スペースキー**を押すと次の対局（次局）が始まります。
